Documentation

EDIT MODE — Modifies the garment directly. Best for color changes, fit adjustments on existing model shots.

RENDER MODE — Produces a clean 3D ghost mannequin product shot on white background. Use "Render As Is" to convert without changes.

SKETCH MODE — Converts garment to a technical flat sketch (CAD style). Auto-sets white colorway.

SALES SHEET MODE — Processes a full page of sketches and converts all to 3D renders simultaneously.

PANTONE CODES — Enter codes like "19-4052 TCX" or names like "Classic Blue". The system resolves to exact hex values.

HISTORY — Select up to 4 renders and click Compare for side-by-side comparison. Use Batch Export to download all versions.
